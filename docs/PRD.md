# 게임 기능 명세

---

# (A) 국가 위치 찾기

### Level 1 (3D 지구본)
- 검은 배경 + 원 형태(또는 캔버스) 3D 지구본
- 질문: 국가 이름이 표시됨
- 사용자는 지구본을 돌려 해당 국가를 클릭
- 정답이면 포인트 누적
- 총 3번 시도 후 계속 틀리면 게임 종료


### Level 2 (수도 맞추기)
- 질문: 수도 이름이 표시됨
- 사용자는 먼저 “해당 수도가 속한 국가”를 고름(국가 선택 단계)
- 맞으면 해당 국가의 2D 지도가 표시되고,
- 지도에서 수도 위치를 클릭하면 됨(수도 클릭 단계)

---

# (B) 국가 인구수 맞추기

### Level 1
- 국가 3개가 카드(박스)로 표시
- 사용자는 인구수 많은 순서대로 나열(드래그/버튼 방식 어느 쪽이든 가능)
- 서버가 정답 체크 후 결과 표시

### Level 2
- 국가 5개로 확장
- 나머지 규칙 동일



---

# (C) AI 국가 추천 (선택)
- MBTI처럼 8~12문항 설문(객관식)
- 백엔드가 설문 응답을 구조화 JSON으로 만들고 LLM API 호출
- “가장 싼/가벼운 모델”을 쓸 수 있도록 LLM Provider는 설정으로 교체 가능하게
- 결과: 추천 국가 Top3 + 이유 + (가능하면) 적합도 점수
